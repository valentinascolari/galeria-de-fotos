 <h1> Galeria de Fotos </h1> <br>
  <h2> Descrição </h2> <br>
    Aplicação web desenvolvida como exercício de programação,
    cujo objetivo é exibir uma galeria de fotos com funcionalidade
    de busca por palavras-chave. As imagens são apresentadas em
    layout grid e a aplicação se adapta a diferentes tamanhos
    de tela, garantindo responsividade.

Funcionalidades:
  - Exibição de galeria de fotos em layout grid
  - Busca em tempo real pelo nome das fotos
  - Filtro por palavras-chave normalizadas
  - Mensagem informativa quando nenhuma foto é encontrada
  - Efeito visual ao passar o mouse sobre as imagens
  - Layout responsivo para dispositivos móveis e desktop

<h2>Tecnologias Utilizadas</h2>
  - HTML5 <br>
  - CSS3 <br>
  - JavaScript (ES6) <br>

<h2> Como rodar localmente </h2>
    - Baixar ou clonar o repositório do projeto; <br>
    - Acessar a pasta raiz do projeto; <br>
    - Abrir o arquivo index.html diretamente no navegador. <br>

<h2> Responsividade </h2>
    A aplicação foi desenvolvida utilizando CSS Grid,
    media queries e a meta tag viewport, garantindo
    adaptação adequada em diferentes tamanhos de tela
    e orientações (desktop, tablet e dispositivos móveis).

<h2>Observações </h2> <br>
  - As imagens utilizadas no projeto são apenas para fins educacionais; <br>
  - A busca considera apenas o nome das fotos. <br>
